# channels/example-vn/static/

Static brand assets for this channel.

- `avatar.png` (or `.jpg`) — optional round avatar shown top-left. Reference
  it from `channel.env`:

  ```
  AVATAR_FILENAME=avatar.png
  ```

- `logo.png` — optional brand logo for HeroCardWithLogo scenes.

Recommended sizes:

- Avatar: 360x360 PNG (square, transparent background)
- Logo:   square ≥ 256px or wide ≥ 800x256
